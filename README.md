# Aesthetic Pomodoro Timer

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## Deskripsi

**Aesthetic Pomodoro Timer** adalah aplikasi manajemen waktu berbasis web yang dirancang dengan antarmuka **Glassmorphism** yang modern dan estetik. Aplikasi ini membantu pengguna meningkatkan produktivitas dengan menggunakan teknik Pomodoro (25 menit kerja, 5 menit istirahat).

## Teknologi

- **Frontend**: HTML5, Modern CSS3 (Animation, Glass Effect), Vanilla JavaScript (ES6+).
- **Backend / Server**: Node.js & Express (untuk serving static files).

## Cara Menjalankan

1.  **Clone repository**:

    ```bash
    git clone https://github.com/NXRts/Pomodoro_Time.git
    cd Pomodoro_Time
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    ```

3.  **Jalankan server**:

    ```bash
    npm start
    ```

4.  Buka browser dan akses: `http://localhost:3001`

## Struktur Project

- `public/`: File frontend (HTML, CSS, JS, Assets).
- `index.js`: Server backend sederhana menggunakan Express.
